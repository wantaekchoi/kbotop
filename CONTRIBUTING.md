# 기여 안내

kbotop에 관심 가져 주셔서 고맙습니다. 작은 프로젝트라 절차는 가볍습니다.

## 시작하기 전에

- 버그 제보나 기능 제안은 **이슈를 먼저** 열어 주세요. 큰 변경일수록 방향을 먼저 맞추는 게 서로 낫습니다.
- 사소한 수정(오탈자·문서·명백한 버그)은 바로 PR을 열어도 됩니다.

## 개발

```sh
cargo test
cargo clippy --all-targets -- -D warnings
cargo fmt
```

세 가지가 모두 통과해야 합니다(CI도 같은 걸 봅니다). 최소 지원 러스트 버전은 `Cargo.toml`의 `rust-version`을 따릅니다.

- 커밋 메시지는 `feat:` / `fix:` / `docs:` / `chore:` / `test:` 접두사를 씁니다(변경 이력이 이 접두사로 정리됩니다).
- 동작을 바꾸는 변경에는 테스트를 함께 넣어 주세요. UI 변경은 "기대한 게 전부 렌더되는지"를 검증합니다.
- 문서는 한국어 우선입니다. `README.md`(한국어)를 고치면 `README.en.md`(영어)도 맞춰 주세요.

## 데이터에 대해

경기·순위·중계 데이터는 네이버 스포츠의 공개(비공식) 엔드포인트에서 옵니다. 파싱은 **관용적으로**(필드가 없거나 null이어도 전체가 깨지지 않게), 그리고 **패닉 없이** 처리하는 것이 원칙입니다.
